BlobCity Database

PROJECT DESCRIPTION

BlobCity offers a high performance flat file database which provides SQL support for applications while also providing horizontal scalability like No-SQL systems.
Users developing on Java EE can leverage the No-Query mode for added performance without client application code changes. Non Java EE clients can use TCP connections or Webservices to communicate with the database.

Soccer Schedule

Soccer Schedule System

PROJECT DESCRIPTION

Back in 2005 when most of India had painfully slow 40 kbit/s connections, finding out when your favourite sports team would play it's next match was painfully difficult. It seemed easier to create a page which would quickly load and provide access to this information.
What started off as clean page providing information went on to provide email and SMS notifications for match schedules for users.
This source has not been touched since 2008 and is possibly quite outdated.
